# 🤖 LogoBuddy – AI-Powered Logo Generator

**LogoBuddy** is a smart logo-making application that uses AI to generate unique logo designs based on user prompts. Built with **Next.js**, **Gemini LLM**, and **Hugging Face**, this tool empowers users to quickly create custom logo ideas with ease and creativity.

---

## 🚀 Features

- 🧠 AI-Powered Prompt Handling with **Gemini LLM**
- 🖼️ Logo image generation via **Hugging Face API**
- ⚡ Built with **Next.js** for fast performance
- 🔥 **Firebase Authentication** for user login
- 📁 Store generated logos using **Firestore**
- 🌐 Responsive UI & smooth UX with **Context API** & **REST API**

---

## 🛠️ Technologies Used

- **Next.js** – Frontend & backend (API routes)
- **Gemini LLM** – Prompt interpretation & logic
- **Hugging Face** – Image generation model  
  `strangerzonehf/Flux-Midjourney-Mix2-LoRA`
- **Firebase Auth** – User authentication
- **Firestore** – Database for saving generated logos
- **Context API** – State management
- **Axios** – For RESTful API calls
